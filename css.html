CSS - Cascading Style Sheets

CSS permet de styliser le contenu HTML. Il utilise des sélecteurs pour cibler des éléments spécifiques et des propriétés pour définir leur apparence.

Voici quelques concepts de base en CSS :

  • Sélecteurs : Ciblent des éléments HTML pour les styliser.
  • Propriétés : Définissent l'apparence des éléments (couleur, taille, marges, etc.).
  • Classes et IDs : Permettent d'appliquer des styles à des groupes spécifiques d'éléments.
  • Boîte modèle : Comprend la marge, la bordure, le rembourrage, et le contenu d'un élément.
