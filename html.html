HTML

HTML (HyperText Markup Language) est le langage de balisage utilisé pour structurer le contenu d'une page web. Apprenez les bases d'HTML pour créer la structure de vos pages.

HTML utilise des balises pour décrire la structure logique d'une page web. Voici quelques balises de base :

  • <html> - La balise racine d'un document HTML
  • <head> - Contient les méta-informations du document
  • <body> - Contient le contenu du document
  • <h1> à <h6> - Balises de titre
  • <p> - Balise de paragraphe
  • <a> - Balise de lien
